### RegMap AI Enterprise

[![Flask](https://img.shields.io/badge/Flask-000?style=flat-square&logo=flask)](https://github.com/sev1425/RegMap-AIV)

[![Offline](https://img.shields.io/badge/AI-100%25%20Offline-brightgreen?style=flat-square)](https://github.com/sev1425/RegMap-AIV)

Offline AI compliance platform for regulated banking. Converts any RBI, SEBI, MCA, or IBA circular into Measurable Action Points — department owner, deadline, evidence requirements, and penalty exposure — in under 4 seconds. Runs entirely on-device via a 17-stage local NLP pipeline. Zero cloud calls at inference.

Built for **SuRaksha Cyber Hackathon 2.0** (theme: Agentic Regulatory Intelligence & Compliance).

| Metric | Result |
|---|---|
| Obligations extracted (148-page RBI circular) | 83 |
| Processing time | < 4 seconds |
| False positive rate | < 5% |
| Network calls at runtime | 0 |

[View project →](https://github.com/sev1425/RegMap-AIV)
